# Exercice 4 - Script d'import

## Installation

1. Copier le répertoire `myimportplugin` dans le répertoire `wp-content/plugins` d'une installation WordPress.

2. Depuis le back-offiche de WordPress, activer le plug-in "My Import Plugin".

## Notes

Je n'ai pas pu aller aussi loin que je l'aurais souhaité sur cet exercice, vous trouverez néanmoins la logique de parsing du JSON et d'import des posts et images dans WordPress à l'intérieur du fichier `functions.php`.
